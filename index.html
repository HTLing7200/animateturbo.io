Abstract

Character animation aims to generate realistic, pose-controllable videos from a static image. Its core challenge lies not only in motion control but in maintaining strict semantic consistency: the generated character's identity, appearance, and style must remain strictly coherent with the reference across all frames. While recent diffusion-based approaches excel at this fidelity, their iterative denoising process creates a critical efficiency bottleneck, hindering real-time applications. The distillation methods introduce a fundamental conflict: rapid, few-step generation often degrades the fine-grained details. To resolve this tension, we introduce AnimateTurbo, an efficient framework that achieves high-quality character animation without sacrificing fidelity. AnimateTurbo introduces two key innovations: (1) ReferenceLCM, a lightweight distillation architecture that establishes a high-speed generation baseline. (2) To counteract the semantic degradation inherent in distillation, we introduce Adversarial Mutual Information (AMI), recasting adversarial learning as a proxy for MI maximization. Extensive experiments demonstrate that AnimateTurbo matches or exceeds state-of-the-art methods in visual quality while requiring only 2–4 inference steps, achieving a 10× acceleration and enabling real-time, high-fidelity character animation.
